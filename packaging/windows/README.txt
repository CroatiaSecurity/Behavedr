Behavedr Agent (Windows)
========================

What this is
------------
Multi-platform behavioral EDR agent (early release).
Console agent that registers platform monitors and runs GIDR core.

Install (recommended)
---------------------
Run:  Behavedr-Setup-*-win-x64.exe
- Installs to Program Files\Behavedr
- Start Menu / optional desktop shortcut
- Uninstall from Windows Settings → Apps

Portable (optional)
-------------------
Zip contains a single Behavedr.exe — run it; no install required.

Run from a terminal
-------------------
  Behavedr.exe

Notes
-----
- Early build: detection hooks are stubs.
- No tray UI yet in this version.
- GitHub: https://github.com/CroatiaSecurity/Behavedr
